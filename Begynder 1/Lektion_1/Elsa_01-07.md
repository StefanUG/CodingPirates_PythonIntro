## Level 7 i Python

Husk hvordan en løkke ser ud

```python
for i in range(???):
```

Husk indrykning!

Og husk du kan bruge Tab og Shift-Tab for at indrykke og rykke ud.

Gør det selv færdigt

```python
from turtle import *

forward(100)
back(100)
right(90)

done()
```

HINT:

```python
for i in range(4):
    forward(100)
    back(100)
    right(90)
```
